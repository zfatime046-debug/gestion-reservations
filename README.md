# 📌 Gestion des Réservations

Projet Java Maven utilisant **JPA (Hibernate)** et la base de données embarquée **H2**.

Ce projet implémente un système simple de gestion de réservations avec les entités :

- 🏢 Salle
- 🛠 Equipment
- 👤 Utilisateur
- 📅 Reservation
- 

## 🧰 Technologies utilisées

- Java 17+
- Maven
- Hibernate (JPA)
- H2 Database
- IntelliJ IDEA
